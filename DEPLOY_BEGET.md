# Деплой на Beget

На Beget можно разместить **только фронтенд** (страницу). Бэкенд (FastAPI) нужно поднять на другом хостинге с поддержкой Python. Ниже — пошаговая схема.

---

## 1. Бэкенд (API) — отдельный хостинг

Beget на тарифе «Сайт» не запускает Python. Нужен хостинг с Python, например:

- **Railway** (railway.app) — есть бесплатный лимит  
- **Render** (render.com) — бесплатный tier  
- **PythonAnywhere** (pythonanywhere.com) — бесплатный аккаунт  

### Вариант: Railway

1. Зарегистрируйтесь на [railway.app](https://railway.app).
2. New Project → Deploy from GitHub (или загрузите папку проекта).
3. Укажите корень проекта, команду запуска: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`.
4. В Variables (переменные окружения) добавьте:
   - `TELEGRAM_BOT_TOKEN` — токен бота
   - `TELEGRAM_BOT_USERNAME` — имя бота без @
   - `WEBAPP_BASE_URL` — **полный URL вашего сайта на Beget** (см. шаг 3), например `https://ваш-сайт.ru`
   - при необходимости: `JWT_SECRET`, `USDT_WALLET_TRC20`
5. После деплоя скопируйте URL приложения (например `https://matrix-xxx.railway.app`). Он понадобится для фронта на Beget.

---

## 2. Фронтенд на Beget

1. Войдите в **панель Beget** → Файловый менеджер (или подключитесь по FTP).
2. Откройте папку **public_html** вашего сайта (или поддомена).
3. Загрузите в **public_html** два файла из папки **deploy/beget/**:
   - **index.html**
   - **config.js**
4. Откройте **config.js** в редакторе на Beget и укажите адрес бэкенда:
   ```javascript
   window.MATRIX_API_BASE = 'https://ваш-бэкенд.railway.app';
   ```
   Подставьте свой URL (без слэша в конце). Если бэкенд будет на другом домене — укажите его.
5. Сохраните файл.

Проверка: откройте в браузере `https://ваш-сайт.ru` (или ваш домен на Beget). Должна открыться страница MATRIX. Если в консоли браузера (F12) есть ошибки запросов к API — проверьте `MATRIX_API_BASE` в config.js и CORS на бэкенде (в проекте уже настроен `allow_origins=["*"]`).

---

## 3. Бот и ссылка на приложение

1. В **.env** на компьютере (или в переменных окружения бэкенда) задайте:
   ```env
   WEBAPP_BASE_URL=https://ваш-сайт.ru
   ```
   Тот же адрес, по которому открывается страница на Beget (с https, без слэша в конце).
2. Перезапустите бота: `python3 bot.py`.
3. В BotFather (или в настройках бота) укажите URL Mini App / веб-приложения: `https://ваш-сайт.ru` (если требуется).

После этого кнопка в боте «Открыть веб-приложение» будет вести на вашу страницу на Beget.

---

## 4. Итог

| Что              | Где размещено | URL пример                    |
|------------------|--------------|-------------------------------|
| Страница (фронт) | Beget        | https://ваш-сайт.ru           |
| API (бэкенд)     | Railway/др.  | https://matrix-xxx.railway.app |
| Бот              | У вас / сервер | токен в .env, бот крутится отдельно |

- В **config.js** на Beget: `MATRIX_API_BASE = 'https://...railway.app'`.
- В **.env** бэкенда и бота: `WEBAPP_BASE_URL=https://ваш-сайт.ru`.

Если что-то не открывается или не работает — проверьте: 1) доступность API по его URL в браузере (например `/docs`); 2) что в config.js указан именно этот URL без лишнего слэша; 3) что WEBAPP_BASE_URL в боте совпадает с адресом сайта на Beget.
